# 29/11/2025

> A aula mostra como as empresas de nuvem como AWS e AZURE funcionam nos dias atuais sendo utilizado por empresas. Ao inves de pensarmos sobre cada conceito separado,aprendemos que tudo se mistura pra dar mais segurança, desempenho e praticidade.A ideia é mostrar a responsabilidade compartilhada entre entre usuarios e empresas de nuvem AWS/AZURE. Também foi abordado temas de infraestrutura de nuvens.

> Foi criado uma conta no Microsoft Azure, onde configuramos nossa conta utilizando o e-mail institucional tendo um pequeno saldo para ser utilizado. Tivemos uma pequena introdução sobre as funcionalidades do Microsoft AZURE.
